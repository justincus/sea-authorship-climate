The data and script included in this repository are the supplementary materials for the manuscript titled "Where is Southeast Asian Climate Research Produced? A Bibliometric Analysis of Climate Science Literature" by Custado, MJ and Cooperdock, EHG (2025).

The Python script (‘SEA_data_analysis_script.py’) summarizes data from the compiled dataset. The auxiliary data (‘region.csv’) is needed to group countries via geographical region.
To run the Python script, users must first export the datasheet tabs containing the organized datasets (‘author_data_clean’, ‘funding_statement_data_clean’, and ‘publication_data_clean’) as separate CSV files.
